Welcome to Chatshell 0.1

                this is the first version of Chatshell that allows you to chat with your friends , teamate or anyone you want.
                create your own shell and start chat, or join your friend's shell.
                this website provides you an easy way to comunicate with people as you can search for people you like
                 and invite them to join your shell,
                also you can create as many shell as you want, and choose your members that you like.

Authentication System

what is authentication system in Chatsell?

                it takes nothing to make an account in Chatshell, just your username, password, age and your gender,
                we will take this information for searching purposes in the future updates,
                to make an account in Chatshell make sure that
                your username is uniqe and this helps us to make better search engine performance, 
                also don't put any suspicious text like
                XML injection or database injection, we have a good regex system to detect this.

Shell System

what is the shell system in Chatshell?

               shell is a place where you can comunicate with anyone, you can call it room or club or anything you like.
               after you login you will find a button that has plus on it, this button allows you to create a new shell,
               the popup form will apear automatically after you click on it, then you have to define the name of your shell
               and the capacity of it, which means that how many people can join.

Request System

what is the request system in Chatshell?

                after searching for an accounts and clicked on it you will redirected to profile page of this account 
                and you will see two kind of requests 
                   1- Friend Request :
                        it will send a friend request to the account you picked, 
                        then the account will recieve the request and will accpet or deny it
                   2- Room Request :
                        if you choose room request the popup panel will appear automatically 
                        and you will see your own shells which you created,
                        then you have to pick one to send a request to the account that ask him to join the shell,
                        if the account accept it he will see this 
                        shell in the subscribed room panel at left of the home page.
                        in your shell you will see the account in the member panel 
                        and you can kick him out of the shell if you want as you are the admin of it.

Chat System

what is the chat system in Chatshell?

                we use Ratchet websocket system and it a high level socket programming 
                that allows you to send messages to other connecting people in the server 
                but count on it, the collusion will occur as the message will broadcasting to all connections, 
                so we designed a system to abstract each connection
                with it's shell,to avoid collusion we make a dynamic connection id for each connection
                that connects to the server in a specific shell so that 
                the collusion will be unreachable.
